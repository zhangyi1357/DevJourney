# Markdown 笔记指南

## 图片管理

图片管理一直是 Markdown 笔记的痛点，因为 Markdown 本身不支持存储图片，一般图片都是作为附件存储在项目中或者存储在云端，然后通过链接引用。

本项目排除任何使用云端的方式，例如各种图床服务、对象存储、在线云文档等，而是使用本地存储的方式将图片存储在项目中，然后通过相对路径引用。

## VSCode 插件

VSCode Markdown 插件很多，但是大多数都不太好用，只推荐一个使用体验接近 Typora 的插件：

- [Inkdown](https://marketplace.visualstudio.com/items?itemName=1943time.inkdown)

优点：

1. 高颜值
2. 所见即所得
3. 大纲展示
4. 支持 mermaid 图形语法
5. 支持 [Katex](https://katex.org/) 公式
